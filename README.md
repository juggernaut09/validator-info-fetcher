# validator-info-fetcher for WITVAL 

```
An expressJS server used to fetch validator details from LCD and store it in Redis.
```

## Install dependencies

```
$ npm install
```

## Run redis-server

```
redis-server
```

## Run Node server

Export APP_ENV 
```
export APP_ENV=dev 
or 
export APP_ENV=prod
```


Start the process
```
nodemon 
or 

npm run start 
```
